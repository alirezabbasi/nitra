# BUG-00004: Manual 90-Day Backfill Did Not Enforce Full `1m` Continuity

## Status

- Resolved in code
- Runtime restart required

## Description

Manual chart backfill (`POST /api/v1/backfill/90d`) rebuilt candles only from currently available `raw_tick` rows.
If the database was missing historical ticks or had sparse ranges, the endpoint still returned success-like behavior without enforcing complete 90-day minute continuity.

## Impact

- `ohlcv_bar` could remain incomplete across the required 90-day `1m` window.
- Single missing minutes remained unresolved.
- UI reported backfill completion while coverage was still partial.

## Reproduction

1. Start with `ohlcv_bar` containing less than full 90-day `1m` bars for a symbol.
2. Trigger chart `Backfill 90d`.
3. Observe endpoint only aggregates from local `raw_tick`; unresolved minute gaps remain.

## Root Cause Analysis

- Backfill endpoint had no strict window-completeness check.
- No venue-history fetch path existed for missing minutes not present in `raw_tick`.
- Response semantics did not distinguish full continuity from partial coverage.

## Resolution Implemented

- Added strict 90-day closed-minute window enforcement in `services/charting/app.py`:
  - compute canonical 90-day window,
  - rebuild from `raw_tick`,
  - detect missing `1m` buckets via `generate_series`,
  - fetch missing ranges from venue adapters (`oanda`, `coinbase`),
  - re-check completeness and return explicit `complete_90d_1m` status.
- Updated chart UI status messaging to surface incomplete coverage and missing-minute counts.
- Added charting env wiring for venue history endpoints/tokens in compose.

## Verification Evidence

- Endpoint now returns explicit fields:
  - `complete_90d_1m`
  - `missing_before_fetch_count`
  - `missing_after_fetch_count`
  - `venue_fetch_error`
- UI shows completion only when full continuity is satisfied.

## Notes

- If venue API cannot provide full minute history (or adapter is unavailable), endpoint reports incomplete status explicitly instead of silent partial success.

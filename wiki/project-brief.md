# NITRA Project Brief

NITRA is a deterministic-first AI-native trading infrastructure project.

## Core mission
- Build a production-grade, Docker-first trading platform with deterministic control boundaries.
- Keep architecture, delivery, and operations continuously synchronized.
- Use Eshel as the mandatory development operating system for compounding project intelligence.

## Core constraints
- Deterministic core logic remains fail-closed and test-verifiable.
- LLM and probabilistic components remain advisory and governed.
- Memory, execution, and quality evidence must remain durable and auditable.

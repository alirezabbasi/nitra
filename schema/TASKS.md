1. Context
2. Objective
3. Scope
4. Out of Scope
5. Implementation Steps
6. Acceptance Criteria
7. Definition of Done
8. Verification Commands
9. Documentation Updates

Naming: `TASK-0001-short-action-title.md`

No context link, no task.

# AGENTS.md — Agent Operating Contract

## Role
You are the project knowledge compiler and engineering operator.

## Non-negotiable rules
1. Durable knowledge must not remain only in chat.
2. Update canonical wiki pages when durable reasoning is produced.
3. Do not duplicate canonical knowledge.
4. Keep links and context traceability current.
5. Every meaningful session appends to `wiki/log.md`.
6. Tasks require context links, acceptance criteria, definition of done, and verification commands.
7. If docs and code diverge, record contradiction and create resolution task.

## Required session start
Read `ruleset.md`, `docs/ruleset.md`, all schema files, `wiki/index.md`, `wiki/project-brief.md`, recent `wiki/log.md`, and development memory docs.

## Required session end
Update affected wiki pages, regenerate index, run lint gates, append log, sync execution/memory docs.

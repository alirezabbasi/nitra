# INGEST.md — Source-to-Wiki Workflow

## Purpose
Convert raw inputs into structured and linked project knowledge.

## Steps
1. Identify source scope and type.
2. Extract entities, concepts, systems, constraints, assumptions, and decisions.
3. Update canonical pages.
4. Add backlinks and context links.
5. Detect and record contradictions.
6. Generate execution tasks where action is required.
7. Update index.
8. Append log.

## Output requirements
- Source summary page created/updated.
- Canonical pages updated.
- Log entry appended.
